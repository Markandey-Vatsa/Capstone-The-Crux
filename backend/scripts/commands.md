# Useful Commands

## Backend (Spring Boot)
- cd backend
- mvn spring-boot:run


## Frontend (Static site)
- `start firefox index.html`
  - open any HTML file directly in the browser

## Taxonomy Manager UI (Flask)

This is very helpful - Opens a new web page to help add files from pending to custom taxonomy
- python -m venv .venv
- .\.venv\Scripts\Activate.ps1
- `pip install -r backend\scripts\requirements.txt
- cd backend\scripts\taxonomy_ui
- python app.py
