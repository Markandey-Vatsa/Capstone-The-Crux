## AI-Based News Aggregation Platform

**Overview**
This project is an AI-powered News Aggregation Platform designed to help users consume news more intelligently and interactively. The platform collects news from multiple sources, analyzes articles using AI, and provides tools that allow users to understand, filter, and interact with news content more effectively.

The system includes AI-driven analysis, personalized filtering, and interactive features such as a chatbot that allows users to ask questions about articles.

This repository contains the backend services developed in Java, which handle data processing, API management, and AI feature integration.

**Key Features**

**Bias Analysis**

The platform analyzes news articles to identify potential political or ideological bias. This helps users evaluate how different sources present information and promotes more balanced news consumption.

**Chatbot – Talk to Your Article**

Users can interact with a chatbot that understands the content of a news article. By asking questions, users can quickly extract insights or summaries from the article.

**Reading Lists**

Users can create reading lists to save, organize, and share articles with others. This allows easier tracking of important news.

**Personalized News Filtering**

The platform allows users to filter news according to their preferences, such as topics, interests, or sources, providing a more relevant news feed.
